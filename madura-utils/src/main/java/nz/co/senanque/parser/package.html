This is the generic parser package which is used to parse various things such as rule files and XML files (or strings).

The core class is Parser which is normally extended to provide the specific grammer details of the input files.

You can pass the Parser class an InputStream to parse by extending AbstractTextprovider and passing it an InputStream. The extended AbstractTextProvider is also used as a kind of visitor pattern object to gather details about the parsed file.
